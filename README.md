Clone the project

```bash
  git clone https://github.com/HarshitSeth-809014/ChatApp_NodeJS.git
```

Go to the project directory

```bash
  cd my_project/server
```

Install dependencies

```bash
  npm install
```

```bash
  node server/index.js
```
